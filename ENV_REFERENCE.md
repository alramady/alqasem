# Environment Variables Reference

## Required
- DATABASE_URL - MySQL connection string
- JWT_SECRET - Random secret for session cookies

## SMTP (for email/OTP)
- SMTP_HOST
- SMTP_PORT (default: 587)
- SMTP_USER
- SMTP_PASS
- SMTP_FROM_NAME (default: القاسم العقارية)
- SMTP_FROM_EMAIL

## Optional
- NODE_ENV (production)
- PORT (default: 3000)
